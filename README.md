#   TripManager - מערכת ניהול הרשמה ומיקום עבור טיול בבית הספר 

## מה המערכת עושה?

מערכת לניהול הרשמת תלמידים ומורים, ומעקב אחרי התלמידים בזמן אמת במהלך טיול.
המערכת מחשבת מרחקים בין תלמידים למורה ומציגה התראות במקרה של התרחקות חריגה.

---

## הדפים במערכת:

1.1. דף הבית  
2. דף רישום והתחברות למורה  
3. דף רישום לתלמידה  
4. מפת מעקב.

   * המורה רואה את כל התלמידות שלה על המפה.
   * הורים יכולים לראות את ילדיהם.
   * תלמידות בטווח מוצגות בצבע ירוק.
   * תלמידות מחוץ לטווח מוצגות בצבע אדום.
   * המורה מוצגת בצבע כחול.

---


### איך המיקומים במערכת עובדים:

* המערכת מקבלת נתוני מיקום בפורמט JSON מהתלמידים
* השרת ממיר את הנתונים לפורמט דצימלי ולהפך לפי הצורך

### פונקציה מיקום רנדומלית:

* פונקציה שמגרילה מיקום רנדומלי עבור תלמיד או מורה בעת ההרשמה למערכת, (כאשר המערכת תעבוד ותקבל מיקום ממכשיר GPS יהיה פונקציה המשנה את מיקום התלמידים והמורה)

### פונקציות הרשמה במערכת:

* פונקציות להוספת תלמיד חדש
* פונקציות להוספת מורה חדש

---

## פונקציות הניטור במערכת:

פונקציית הניטור אחראית לעדכון המיקום של כל תלמיד בזמן אמת.

הפונקציה:

* מחשבת מיקום דינמי של המורה והתלמידים
* מבצעת סימולציה של תנועה לפי זמן ומרחק מנקודת מוצא
* מסננת את התלמידות לפי הכיתה של המורה הרלוונטי
* מחשבת את המרחק הפיזי בין כל תלמידה למורה
* בודקת האם התלמידה נמצאת בטווח תקין

אם תלמידה נמצאת במרחק גדול מ־3 קילומטר:

* היא מסומנת כחריגה
* האייקון שלה משתנה לאדום

### איך מריצים את הפרויקט:

להיכנס לתיקיית הפרויקט (client):

cd client

להתקין תלויות:

npm install

פקודת ההרצה:
npm start

כתובת הדפדפן:


http://localhost:3000/


להיכנס לתיקיית השרת:

cd server

שחזור תלויות:

dotnet restore.

יש לוודא שמריצים את הפקודות מתוך התיקיות הנכונות
.
npm install מתקין את כל חבילות ה־ React.

dotnet restore  מוריד את כל חבילות ה־ NuGet .הנדרשות
